picasso plugin for `Tutor <https://docs.tutor.edly.io>`__
#########################################################

picasso plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/eduNEXT/tutor-contrib-picasso

Usage
*****

.. code-block:: bash

    tutor plugins enable picasso
